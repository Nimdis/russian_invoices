# RussianInvoices
Гем для генерации различных документов в формате pdf для rails приложегния.
В данный момент есть возможность генерации следующих видов документов:
* счет-фактура;
*  акт выполненных работ;
*  договор.


## Установка

``` ruby
  gem 'russian_invoices'
```
